## [1.0.0] - 2026-01-26

### 新增

#### 核心功能
- ✨ 实时语音转录功能
  - 支持讯飞医疗语音识别 API
  - 实时显示转录结果
  - 医疗术语准确率 > 95%
  - 支持说话人区分

- ✨ 病例结构化功能
  - 自动识别主诉、现病史、既往史等关键信息
  - 基于讯飞星火大模型智能提取
  - 支持量化描述（体温、血压等）
  - 自动过滤无关信息

- ✨ 病历生成功能
  - 符合病历书写规范的标准化模板
  - 根据结构化数据自动生成
  - 支持手动编辑和补充
  - 专业医学文书格式

- ✨ 文档导出功能
  - 支持 Word 文档导出
  - 标准化病历格式
  - 包含患者信息、医生信息
  - 一键生成专业文档

- ✨ 数据管理功能
  - 本地安全存储病例数据
  - 支持按患者姓名、日期检索
  - 支持数据导出和备份
  - 隐私保护，数据不上传云端

#### 桌面应用
- 🖥️ 基于 tkinter 的直观操作界面

## 版本说明

### 版本号规则

- **主版本号**：不兼容的 API 修改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正

### 变更类型

- **新增** (Added)：新功能
- **变更** (Changed)：现有功能的变更
- **弃用** (Deprecated)：即将移除的功能
- **移除** (Removed)：已移除的功能
- **修复** (Fixed)：问题修复
- **安全** (Security)：安全相关的修复
